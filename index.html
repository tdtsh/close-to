Business

クロストゥは古都京都を拠点に美容系ベンチャーのアプリ・サービス開発・運営、ならびにiPhone/Androidアプリ開発、Node.jsやGoogle App Engine/Javaによるバックンド・Webシステム受託開発、AWSの設計・構築コンサルティングをおこなっています。

iPhone/Androidアプリ開発

Titanium Mobile (Alloy) を用いたクロスプラットフォームなアプリ開発を得意としています。SwiftやJavaによるネイティブアプリ開発もご相談ください。

バックエンド開発

Node.js (Express / Sails.js) を使ったMobileアプリのバックエンドやWebサービス開発承ります。REST API + RDS (MySQL) + Bootstrapを用いた管理画面という組み合わせが比較的得意です。Google App Engine /Java も実績あります。ご相談ください。

クラウド・サーバ構築

クラウド・VPSサービス上に Linuxサーバ (比較的CentOS/Amazon Linux、Ubuntuが得意です)とミドルウェアを構築します。AWSの設計・構築・運用 (VPC、ELB、RDS、Route 53等) もご相談ください。
